Please see [this section of the workshop template][workshop-setup]
for instructions on installing Git.

We'll do our work in the `git-novice` folder, please follow the commands instructed below:

~~~
$ cd ~
$ mkdir git-novice
$ cd git-novice
$ which git
$ git -v
~~~
{: .language-bash}
